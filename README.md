# AI Launchpad by Data Sense

A complete React + Tailwind landing page for the Data Sense AI learning ecosystem.

## Included

- Responsive React landing page
- Tailwind CSS configuration
- Reusable components
- AI Launchpad learning tracks
- Practice Arena preview
- SQL Practice Arena
- SQL Casino and SQL Bureau of Investigation
- Latest learning materials
- Member projects
- Skool and social community section
- Career tools
- Free and Premium pricing
- Generated landing-page reference image in `src/assets/landing-page-reference.png`

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Main files

- `src/App.jsx` — full homepage structure
- `src/data/siteData.js` — editable content and cards
- `src/components/` — reusable UI components
- `src/styles.css` — global styles

## Customize

Update social links, current community counts, events, courses, projects and testimonials in `src/data/siteData.js` and `src/App.jsx`.
